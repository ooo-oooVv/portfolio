메서드 -> 스타일시트 부트스트랩은 그리드 시스템 (스타일의 일관성을 갖고있는 소스 -> 하드코딩 스타일이 적어짐.) EX 1) text-center -> 화면 너비에 상관없이 글자 가운데 정렬 (text-align: center;) text-md-left -> 화면 너비가 768px 이상일 때 글자는 왼쪽 정렬. ㄴ> 모바일일때는 가운데정렬, 태블릿 이상이면 왼쪽 정렬이구나.

우선순위는 반응형 스타일이 더 우선.

EX 2) text-md-center -> 모바일 사이즈는 왼쪽 정렬, 테블릿 이상이면 가운데 정렬. (기본 정렬이 left임.) EX 3 py-md-5 py-2 -> 위아래 padding이 모바일일땐 2(rem), 태블릿이상일때눈 5(rem)
